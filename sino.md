## Coding & Development

### AI Editors & IDEs
* **[Cursor](https://cursor.sh)** - VS Code fork with built-in AI that understands your entire codebase.
* **[GitHub Copilot](https://github.com/features/copilot)** - The industry standard AI pair programmer.
* **[Windsurf](https://codeium.com/windsurf)** - IDE by Codeium with "Cascade" agent for deep context awareness.
* **[Replit](https://replit.com)** - Cloud IDE with autonomous AI agents for project generation.
* **[Tabnine](https://www.tabnine.com)** - Privacy-centric code completion for enterprise.

### Development Frameworks
* **[LangChain](https://python.langchain.com)** - Framework for developing applications powered by LLMs.
* **[LlamaIndex](https://docs.llamaindex.ai)** - Data framework for connecting custom data to LLMs.
* **[Vercel AI SDK](https://sdk.vercel.ai)** - Library for building AI-powered user interfaces.
* **[Ollama](https://ollama.com)** - Get up and running with large language models locally.

## Open Source Models

Access, deploy, and fine-tune these models.

### Large Language Models (LLMs)
* **[Llama 3](https://ai.meta.com/llama)** (Meta) - The current state-of-the-art for open weights.
* **[Mistral / Mixtral](https://mistral.ai)** (Mistral AI) - High performance, efficient open models.
* **[Gemma](https://ai.google.dev/gemma)** (Google) - Lightweight open models based on Gemini research.
* **[Qwen](https://github.com/QwenLM/Qwen)** (Alibaba) - Strong performance in coding and math benchmarks.

### Small / Local Models
* **[Microsoft Phi](https://www.microsoft.com/en-us/research/project/phi)** - High capability in small parameter counts (runs on standard hardware).
* **[TinyLlama](https://github.com/jzhang38/TinyLlama)** - Compact model suitable for edge devices.

## Creative Tools (Video, Audio, Image)

* **[Midjourney](https://www.midjourney.com)** - High-fidelity artistic image generation.
